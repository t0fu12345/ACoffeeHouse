# ACoffeeHouse POS & Admin Management System

📌 **Live Demo:** [https://t0fu12345.github.io/ACoffeeHouse/](https://t0fu12345.github.io/ACoffeeHouse/)

> **Lưu ý quan trọng (Important Note):** Đây là một bài tập thực hành cá nhân (practice project) tập trung hoàn toàn vào việc xây dựng và tối ưu trải nghiệm **UI/UX**. Hệ thống được thiết kế giao diện theo phong cách hiện đại (Modern Glassmorphism) nhằm mô phỏng một phần mềm quản lý quán cà phê chuyên nghiệp.

## 🔐 Hướng dẫn đăng nhập (Login Instructions)

Để có thể truy cập và trải nghiệm giao diện quản trị viên (Admin Dashboard), bạn cần sử dụng thông tin đăng nhập dành riêng cho Admin như sau:

- **Tên đăng nhập (Username):** `admin`
- **Mật khẩu (Password):** `123`

*(Nếu bạn nhập bất kỳ thông tin hợp lệ nào khác ngoài thông tin trên, hệ thống sẽ tự động chuyển hướng bạn đến giao diện Bán hàng (POS) dành cho nhân viên).*

## 🚀 Công nghệ sử dụng
- **Frontend Framework:** React.js (Vite)
- **Styling:** Tailwind CSS v3 (Custom Design System, Variables)
- **Routing:** React Router v7 (HashRouter config tương thích với GitHub Pages)
- **Deployment:** GitHub Actions & GitHub Pages

## 🎨 Điểm nhấn UI/UX
- **Dark Mode & Warm Tone:** Sử dụng tông màu tối kết hợp với màu nâu cà phê espresso mang lại cảm giác ấm cúng, sang trọng.
- **Glassmorphism:** Hiệu ứng kính mờ (backdrop-blur) ở các hộp thoại, sidebar, và thẻ thông tin giúp giao diện có chiều sâu.
- **Layout Management:** Bố cục chia rõ ràng giữa màn hình Bán hàng (POS) linh hoạt và màn hình Quản trị (Admin) đầy đủ chức năng CRM/Inventory.
